The Signup Booster increases signups on your WordPress site using predictive analytics.

== Description ==

The Signup Booster increases signups on your WordPress site using predictive analytics. It works by tracking which web page leads to the highest probability of signup from the current page, and then recommends these high-probability page to your users. 

== Installation ==

Getting Payboard set up is easy. Just install the plugin, request your Payboard API Key, enter the Payboard API Key on the plugin settings page, and you are done.
